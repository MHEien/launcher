## Features

- **25-minute focus sessions** with automatic tracking
- **Smart breaks** - short (5 min) and long (15 min after 4 pomodoros)
- **Visual timer widget** with circular progress indicator
- **Statistics tracking** - daily and weekly stats
- **Command triggers** for quick access
- **AI integration** for hands-free control
- **Desktop notifications** when sessions complete

## Commands

Type `pomo:` in the launcher to access all commands:

| Command | Description |
|---------|-------------|
| `pomo:start` | Start a 25-minute focus session |
| `pomo:break` | Start a break (auto-detects short/long) |
| `pomo:pause` | Pause/resume the current timer |
| `pomo:skip` | Skip to the next phase |
| `pomo:reset` | Reset the timer |
| `pomo:stats` | View today's statistics |

### Pomodoro Stats (2x1, 2x2)
Statistics dashboard showing:
- Today's completed pomodoros
- This week's total
- Total focus time
- Weekly bar chart

## The Pomodoro Technique

1. **Work** for 25 minutes with full focus
2. **Short break** for 5 minutes
3. **Repeat** steps 1-2 three more times
4. **Long break** for 15-30 minutes after 4 pomodoros
